## Текстовое описание решения

Данная программа из имеющегося массива строк формирует новый массив из строк, длина которых меньше, либо равна 3 символам. Программа выполняется с помощью подпрограмм.

### Алгоритм работы программы:
Массив строк задается непосредственно в программе (*строка 39*).
Далее программа обращается к подпрограмме GetPrintArray, передавая в нее массив строк (*строка 40*).

Подпрограмма GetPrintArray (*строки 28 - 34*):
- выводит в терминале элементы массива через пробел в одну строку (*строка 32*).

Затем программа обращается к подпрограмме GetArrayOfThreeCharacters, передавая в нее заданный массив.

Подпрограмма GetArrayOfThreeCharacters (*строки 11 - 24*):
- принимает массив строк (*строка 11*);
- создает пустой массив resalt с длинной, равной длине принятого массива (*строка 13*);
- цикл, который проходит по каждому элементу принятого массива, увеличивая индекс на каждом круге цикла (*строка 15*);
- на каждом круге цикла, если длина элемента массива меньше или равна трем, то в массив result на первый элемент помещает данный элемент принятого массива. Индексы массива result заданы переменной j (*строка 14*), которая увеличивается после заполнения элемента (*строка 20*);
- по завершении цикла программа возвращает массив строк (*строка 23*).

Затем программа снова обращается к подпрограмме GetPrintArray, передвая новый сформированный массив, чтобы вывести его в терминале.
